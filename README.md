# Vibe-learning

> Learn diffusion models from scratch with Claude as your instructor.

This repository pairs a structured exercise guide with a personalized `CLAUDE.md` so Claude Code teaches *you* — adapting to your background, pacing, and learning style.

## Quickstart

```bash
uv sync
```

## Make it yours

| File | Purpose |
|------|---------|
| `CLAUDE.md` | Tell Claude what you know, how you learn, and what to skip |
| `diffusion_models_learning_guide.md` | Adjust exercises, difficulty, and scope to match your level |
